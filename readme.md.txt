# 📝 Blog CRUD Application – ApexPlanet Internship (Task 2)

This project is a basic blog-style CRUD application built using **PHP** and **MySQL** as part of Task 2 in my Web Development Internship at **ApexPlanet Software Pvt. Ltd.**

## 🚀 Features
- ✅ User Registration & Login (with password hashing)
- 📥 Add Blog Posts
- 📄 View Posts
- 📝 Edit & Update Posts
- ❌ Delete Posts
- 🔐 Session-based user authentication

## 🧩 Technologies Used
- PHP
- MySQL (port: 3307)
- HTML & CSS (custom styled)
- XAMPP (local environment)
- Git & GitHub
